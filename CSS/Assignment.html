Shopping Cart Web Page ~~~~~~~~~~~~~~~~~~~~~~~~ Objective: Create a simple Shopping Cart web page using HTML and CSS that displays products and allows users to view selected items in their cart. Page Structure (HTML): * Create a main page titled “My Shopping Cart”. * Display at least 5-6 products. * Each product should include: * Product image * Product name * Short description * Price * A “Add to Cart” button (only for design, no JavaScript needed). 2.
